# tailor-marketplace-frontend
Tailors marketplace web appliction

## Project structure

- `index.html` and the files in `pages/` provide the individual pages.
- `assets/js/layout.js` contains the shared header, footer, and navigation.
- `assets/js/pages.js` stores page-specific content so each screen is easy to update later.
- `assets/css/styles.css` contains the responsive design system for the shared layout.
